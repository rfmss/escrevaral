# Cláudio Mendonça

Escritor de ficção científica, 47 anos. Três romances publicados por editora independente, dois em andamento. Usou Scrivener por 8 anos. Migrou para o Escrevaral por curiosidade e porque o Scrivener em Linux "nunca funcionou direito".

Não é um usuário iniciante. Tem opinião formada sobre como um editor de texto deve se comportar.

## O que ele traz do Scrivener

Cláudio sabe o que um escritor profissional espera de uma ferramenta:

| Hábito do Scrivener | O que ele vai procurar no Escrevaral |
|---|---|
| Binder (árvore de cenas/capítulos) | Acervo com estrutura hierárquica ou pelo menos pastas |
| Compilar para EPUB/DOCX | Exportação com formatação limpa |
| Notas de pesquisa separadas do manuscrito | Notas avulsas ou área de rascunho |
| Metas de palavras por sessão | Cronograma / contagem integrada |
| Backup automático com versionamento | Cópia de segurança + histórico de versões |
| Modo de composição (tela cheia) | Modo foco / modo leitura |

## O que ele vai estranhar

- **Ausência de estrutura de projeto**: Scrivener organiza capítulos como documentos separados. Escrevaral tem manuscritos independentes, não partes de um todo.
- **Exportação**: vai querer DOCX ou EPUB, não só TXT. Vai testar o que sai.
- **Busca dentro do acervo**: vai querer procurar uma palavra em todos os manuscritos.
- **Ausência de notas de pesquisa**: onde coloca referências, links, esboços que não são o texto?

## Como ele usa o produto

1. Abre o Escrevaral. Olha o acervo primeiro, não o editor.
2. Cria dois manuscritos separados — um "Capítulo 1", um "Notas".
3. Tenta exportar o "Capítulo 1" para ver o formato.
4. Acessa o Cronograma para registrar a sessão.
5. Acessa Prova de Autoria para entender o que é aquilo.
6. Usa o Espelho de Voz e compara com o que sente sobre seu próprio estilo.

## O que ele vai sentir (provável)

- **Acervo**: satisfeito com a lista, frustrado com a ausência de agrupamento por projeto.
- **Exportação**: surpreso se funcionar bem; frustrado se só sair TXT.
- **Prova de Autoria**: curioso, mas vai comparar com o versionamento do Scrivener.
- **Espelho de Voz**: genuinamente interessado — é algo que o Scrivener não tem.
- **RimaLab e Decolonial**: vai achar específico demais para ficção científica, mas vai explorar.

## Achados que ele provavelmente vai trazer

- Falta de busca global no acervo
- Exportação insuficiente para fluxo editorial profissional
- Ausência de agrupamento de manuscritos por projeto/série
- Modo foco funciona, mas falta "modo composição" com barra de progresso visível
